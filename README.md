# 🎌 بوت واتساب لتحميل الأنمي المترجم

نظام متكامل لتحميل حلقات الأنمي المترجم من أكثر من **55 موقعاً** مختلفاً وإرسالها مباشرة عبر واتساب.

## ✨ الميزات الرئيسية

✅ **55+ موقع أنمي مدعوم**
- مواقع عربية: WitAnime, Animeiat, Anime4up, وغيرها
- مواقع إنجليزية: Crunchyroll, 9anime, Gogoanime, وغيرها
- مواقع إسبانية: AnimeFlv, Jkanime, وغيرها

✅ **جودات متعددة**
- 360p, 480p, 720p, 1080p, 1440p, 4K

✅ **سيرفرات تحميل متنوعة**
- Mega, MediaFire, Google Drive, Dropbox, وغيرها

✅ **ضغط تلقائي للفيديو**
- يضغط الملفات تلقائياً لتناسب حد واتساب (16 MB)
- خيارات جودة مختلفة

✅ **إدارة الجلسات**
- كل مستخدم له جلسته الخاصة
- سجل كامل للتحميلات
- قاعدة بيانات SQLite محلية

✅ **سهل الاستخدام**
- واجهة نصية بديهية
- تثبيت تلقائي لجميع المتطلبات
- أمر تشغيل واحد فقط

## 💻 الملفات البرمجية

### `run.py`
- نقطة الدخول الرئيسية
- يتحقق من الإعداد
- ينفذ التثبيت التلقائي إذا لزم الأمر
- يشغل البوت الرئيسي

### `whatsapp_bot.py`
- بوت واتساب الرئيسي
- يدير الجلسات والتفاعلات
- يعالج رسائل المستخدمين
- يرسل الملفات

### `anime_downloader.py`
- يبحث عن الأنمي
- يستخرج روابط الحلقات
- يحمل الملفات
- يتعامل مع أخطاء الشبكة

### `video_compressor.py`
- يضغط ملفات الفيديو
- يحول لصيغة متوافقة
- يستخرج صور مصغرة
- يدعم تحديد الجودة

### `database.py`
- إدارة قاعدة بيانات SQLite
- إدارة الجلسات
- تسجيل التحميلات
- حفظ تاريخ المستخدمين

### `anime_sites.py`
- قائمة 55+ موقع أنمي
- معلومات كل موقع
- قائمة السيرفرات
- قائمة الجودات

### `config.py`
- إعدادات البوت الشاملة
- رسائل متعددة اللغات
- إعدادات الضغط
- إعدادات المتصفح
